## Rodando o aplicativo

- Clonar o projeto
- Abrir um terminal da pasta raiz
- Rodar o comando para atualizar as dependências

```
flutter pub get
```
- Abrir seu emulador de celular e depois rodar o comando no terminal
```
flutter run
```
